用 Figma Variables 搭了一套完整的县城设计系统，真的救活了产品团队

去年底开始接手了一个县城级政务+生活服务的产品，团队小，设计师就我一个，但要同时维护 App、小程序、H5 三端。

之前每次换色换间距都要改几百个地方，噩梦。后来痛定思痛，花了三周时间用 Figma Variables 搭了一套三级 Token 映射的设计系统。

L1 是原始值（hex），L2 是语义层（primary/solidBg 这种），L3 是组件层（primary/bt/solidBg）。换主题色只要改 L1 那一层，下面几百个组件全自动更新，属实爽到飞起。

最近把这套系统整理成了设计系统浏览器，放到 GitHub Pages 上给全团队用，顺便还训练了个 AI 助手帮 PM 画原型……
